### Mini Social Network

A mini social network with basic features such as:

* User signup

* CRUD on post

* CRUD on comment

* Like/Unlike a post/comment


__Tools__

* Java Servlet

* JSP

* JUnit

* SQL

* JDBC


__User Stories__

A user should be able to

* Signup

* create a post, comment.

* Like a post or comment

* Unlike a post or comment

* View a post and its comment
